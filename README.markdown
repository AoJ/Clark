Clark
=====

A coffeescript, npm-ready version of the [spark](https://github.com/holman/spark)
shell script. See spark documentation for details how to use.

Requires underscore.

Installation
------------

npm install clark -g

Usage
-----

    clark 1 2 3 4 5

output: ▁▁▃▅▇

Checkout the spark page for more usages.
